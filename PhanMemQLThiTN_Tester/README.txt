SET UP

Mở database SQLData -> Run(F5)

Step 2:
Mở database QUANLYTK -> Run(F5)

Step 3: 
Connect database vào Visual Studio

**Đổi tất cả đường dẫn Data source thành đường dẫn của SQLData || Riêng Form1.cs (XD Form) đổi đường dẫn Data Source thành đường dẫn của QUANLYTK
